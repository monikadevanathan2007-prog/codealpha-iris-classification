# 🌸 Iris Flower Classification — CodeAlpha Internship Task 1

A complete machine learning pipeline to classify Iris flower species
using 5 classification algorithms with full EDA and visualization.

## 📊 Dataset
- 150 samples · 3 species · 4 features
- Source: UCI Machine Learning Repository
- Features: SepalLengthCm, SepalWidthCm, PetalLengthCm, PetalWidthCm
- Target: Species (Iris-setosa, Iris-versicolor, Iris-virginica)

## 🤖 Models & Accuracy
| Model | Accuracy |
|-------|----------|
| K-Nearest Neighbours | 100.0% 🏆 |
| Logistic Regression | 96.7% |
| SVM (RBF Kernel) | 96.7% |
| Decision Tree | 93.3% |
| Random Forest | 90.0% |

## 🛠️ Tech Stack
- Python 3.x
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

## 🚀 How to Run
pip install pandas numpy scikit-learn matplotlib seaborn
python iris_classification.py

## 📈 Key Findings
- KNN achieved perfect 100% accuracy on test set
- Petal features are the strongest species predictors
- Iris-setosa is completely linearly separable
